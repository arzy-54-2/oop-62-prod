Команды для регистрации ваших данных
git config --global user.name "Your name"
git config --global user.email "your-email@example.com"


Команды только для одно проекта
git init
git remote add origin https://github.com/mr-arzybek/oop-62-2.git



git status                | Состояние проекта
git add .                 | Выделяет все и копирует в буфер обмена
git commit -m "lesson1"   | Сохраняет все то что мы скопировали
git branch                | Показывает наши ветки
git push orign master     | Отправляет все наши сохранения или коммиты
